---
description: Deploy the project to a new GitHub repository
---

1. Initialize Git repository
   - Run `git init`
   - Run `git add .`
   - Run `git commit -m "Initial commit"`
   - Run `git branch -M main`

2. Create and push to Remote
   - Ask user for the remote repository URL
   - Run `git remote add origin <REMOTE_URL>`
   - Run `git push -u origin main`
